1. Zainstaluj przez terminal te komendy:
pip install pyfiglet
pip install termcolor
pip install colorama

apt-get install python
2. Wpisz w terminalu:
cd Atak-DOS
python dos.py
3. Wpisz IP Ofiary, Port i odczekaj kilka sekund, 
a potem ciesz sie atakiem DoS!
